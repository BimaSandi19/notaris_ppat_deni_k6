TROUBLESHOOTING
================================================================================

PROBLEM: Tabel tidak format dengan baik
SOLUSI: 
- Delete tabel, re-insert menggunakan Insert > Table
- Atau copy dari Excel (lebih reliable)

PROBLEM: Numbers tidak cocok dengan log file
SOLUSI:
- Re-verify dari file log: d:\Applications\websitedn-k6\results\...
- Check last 100 lines untuk summary statistics

PROBLEM: Grammar/Spelling errors
SOLUSI:
- Use Word's spell checker (Review > Spell Check)
- Gunakan Grammarly online checker (free version)

PROBLEM: Formatting messed up setelah copy-paste
SOLUSI:
- Use Paste Special (Ctrl+Shift+V)
- Choose "Unformatted Text"
- Re-apply formatting manually

PROBLEM: Can't find summary statistics
SOLUSI:
- Buka log file di text editor
- Search untuk "THRESHOLDS" dan "TOTAL RESULTS"
- Copy dari section itu
